# Portfolio Monte Carlo App

This is a Vercel-ready static deployment package for the Portfolio Monte Carlo app.

## What is included

- `index.html` — the app entry point
- `vercel.json` — static deployment configuration for Vercel

## Deploy on Vercel

### Option 1: Drag-and-drop

1. Go to [Vercel](https://vercel.com/).
2. Create a new project.
3. Upload this folder or import it from a Git repository.
4. Vercel will detect it as a static site and deploy it.

### Option 2: Git-based deployment

1. Put these files into a Git repository.
2. Push the repository to GitHub, GitLab, or Bitbucket.
3. Import the repository into Vercel.
4. Deploy with the default settings.

## Notes

- This app is fully client-side and does not require a backend.
- CSV uploads are processed in the browser.
- No database or server runtime is required.
- If you later add authentication or secure file storage, you may want a more controlled deployment architecture.
